# 성숙도 프레임워크 (L1–L5)

4차원 가중평균으로 리포지터리 성숙도를 평가한다.

## 차원과 가중치
| 차원 | 가중 | 구성 원칙 |
|---|---:|---|
| 문서/맵 | 0.30 | P1, P2, P10, P12 |
| 아키텍처/강제 | 0.30 | P3, P4, P6 |
| 품질/테스트 | 0.25 | P5, P9, P7 |
| 안전/운영 | 0.15 | P8, P11 |

## 등급
| 등급 | 가중평균 | 의미 |
|---|---|---|
| L1 | <1.5 | 기록 없음, 진입 문서 부재 |
| L2 | 1.5–2.4 | 문서 일부, 강제 없음 |
| L3 | 2.5–3.4 | 맵·아키텍처 문서 + 일부 자동 검증 |
| L4 | 3.5–4.4 | verify 자동화 + 테스트 게이트 + 안전 |
| L5 | ≥4.5 | 자가 발전 루프, 실측 계약, 완전 자동 검증 |

- 아키텍처/강제: layer-rules + verify-docs → 중상

- 추정 등급: **L4** 초입. 정밀 채점은 `/sh:harness-gc`로.
